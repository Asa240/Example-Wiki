Returning an Order

Welcome to the Order Support section. Here, you'll find all the information you need to return an order.

Step-by-Step Guide to Returning an Order

  1. Locate Your Order
    • Go to your account dashboard.
    • Click on "Order History" to find the order you wish to return.
  2. Check Return Eligibility
    • Ensure that your order is within the return period (usually 30 days from the delivery date).
    • Make sure the items are in their original condition, with all tags and packaging intact.
  3. Initiate a Return
    • Click on the order you wish to return.
    • Select the items you want to return and the reason for the return.
    • Follow the on-screen instructions to generate a return label.
  4. Prepare Your Return Package
    • Print the return label and attach it to the package.
    • Include the original packing slip inside the package.
  5. Ship Your Return
    • Drop off the package at the designated shipping location or schedule a pick-up, if available.
  6. Track Your Return
    • Use the tracking number provided to monitor the return shipment.
    • Once received and processed, you will be notified of the refund or exchange.

If you have any questions or need further assistance, please contact our support team.
